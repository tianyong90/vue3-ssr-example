# vue3-ssr-example

vue3 ssr example, no vite-ssr or vite-ssr-plugin used

## Features

- [x] typescript
- [x] tailwindcss 
- [x] eslint 
- [x] stylelint 
- [x] router 
- [x] pinia(store) 
- [x] vueuse/head 
- [x] svg sprite 
- [x] sitemap
